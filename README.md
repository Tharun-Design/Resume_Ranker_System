#  Resume Ranker System
**A Multi-Resume ATS Ranking Application powered by Google Gemini 2.5 Flash.**

This project is a powerful Applicant Tracking System (ATS) application that streamlines the hiring process by intelligently analyzing and ranking multiple candidate resumes against a specific job description.

---

## Features

* **Multi-Resume Upload:** Upload multiple PDF resumes simultaneously for bulk evaluation.
* **Accurate PDF Processing:** Utilizes the **Gemini File API** for robust and accurate content extraction from PDF documents.
* **Strict JSON-based Evaluation:** Employs precise, structured JSON output from Gemini for ATS evaluation, ensuring objective and consistent scoring.
* **Automatic Ranking:** Automatically ranks resumes based on relevance to the job description.
* **Detailed Breakdown:** Provides comprehensive reports for each resume, including:
    * **Match Percentage**
    * **ATS Score**
    * **Skills Missing**
    * **Strengths**
    * **Weaknesses**
    * **Summary Report**
* **Modern Interface:** Built with a user-friendly and responsive Streamlit interface with improved UI elements.
